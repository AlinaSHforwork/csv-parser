# CSV to PostgreSQL Stream Parser

### Node.js web application that scans a local folder for CSV files and streams them into a PostgreSQL database.

---
## Set up:
- Install Dependencies:

```bash
npm install
```
- Database Config:

```.env
DB_URL=
```
## Start server: 

```bash
npm start
```

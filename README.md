# CSV to PostgreSQL Stream Parser

### Node.js web application that parses CSV files and streams them into a PostgreSQL database.

---

## Set up:

- Install Dependencies:

```bash
npm install
```

- Database Config:

```.env
DB_URL=
```

## Start server:

```bash
npm start
```

## Also added Sequelize version of parser, to run:

```bash
npm run startSequelize
```
